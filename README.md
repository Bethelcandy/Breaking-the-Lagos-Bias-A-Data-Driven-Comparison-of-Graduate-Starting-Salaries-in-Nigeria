# Breaking-the-Lagos-Bias-A-Data-Driven-Comparison-of-Graduate-Starting-Salaries-in-Nigeria
Breaking the Lagos Bias: Graduate Salaries in Nigeria compares Lagos, Kano, and Edo using Python and 907 records. Analysis of p-values (0.923; 0.388) proved no significant difference in average pay. Kano's 90th percentile (₦333,852) even exceeded Lagos (₦325,658), proving consistent pay across hubs.
